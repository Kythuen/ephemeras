# Linter
![](https://img.shields.io/codecov/c/github/Kythuen/ephemeras?flag=linter)

## 简介
@ephemeras/linter 是一个命令行工具，用于快捷配置和管理项目的代码规范。

## 安装

```shell
pnpm install -g @ephemeras/linter
```
