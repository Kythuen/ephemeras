# Utils

![](https://img.shields.io/codecov/c/github/Kythuen/ephemeras?flag=utils)

## 简介
@ephemeras/utils 的定位是提供 Node 项目开发过程中常用的一些方法，快速有效地处理相应的需求。

现提供了下列几个模块：
- [validate](/utils/features/validate): 严格的参数校验。
- [build](/utils/features/build): 项目构建相关的一些方法。
- [profile](/utils/features/profile): 配置文件管理。

## 安装

```shell
pnpm add -D @ephemeras/utils
```

## 使用
```ts
import { readPKG } from '@ephemeras/utils'

```
